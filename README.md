# 🚀 Navi Cleaner v5.5 — Suite de Optimización Inteligente

Navi Cleaner es una aplicación de escritorio premium, diseñada bajo la estética limpia y minimalista de Apple, que limpia el almacenamiento de tu computador, optimiza la memoria RAM y organiza tus archivos de forma 100% automática y privada.

## 📥 Descarga Directa (Plug & Play)

Para usar la aplicación, no necesitas instalar herramientas de programación ni ejecutar comandos. Solo haz clic en el siguiente enlace:

👉 **[DESCARGAR NAVI CLEANER PARA WINDOWS](https://github.com/adasmeproyectos/local-agent-monitor/releases/tag/v6.0)**

---

## 🛠️ Cómo usarlo en tu Computador (Paso a Paso)

1. **Descarga el archivo:** Haz clic en el enlace de arriba para descargar `Navi Cleaner.exe`.
2. **Abre el programa:** Haz doble clic sobre el archivo descargado.
3. **Advertencia de Windows (Solo la primera vez):** Al ser un software personalizado creado por ti, Windows Defender mostrará una pantalla azul. Solo debes hacer clic en **"Más información"** y luego en el botón **"Ejecutar de todas formas"**.
4. **¡Listo!** Se abrirá automáticamente una pestaña en tu navegador web con la aplicación en español. Presiona **"Escanear Ahora"** y deja que el sistema organice tu computador.

## 🧠 Cerebro Adaptativo Local
La aplicación es inteligente y se adapta a tu perfil de trabajo de forma privada:
- **Para la Construcción:** Detecta y organiza automáticamente presupuestos, licitaciones, cubicaciones y planos.
- **Para Finanzas:** Agrupa balances, facturas y flujos de caja de forma limpia.
- **Protección Gamer:** Limpia la basura de los launchers (Steam, Epic, Riot) pero jamás toca ni rompe tus videojuegos instalados.

Al cerrar la ventana o presionar el botón **Apagar**, el programa se cierra por completo y libera el 100% de los recursos de tu hardware.
